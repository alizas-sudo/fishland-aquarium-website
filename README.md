# 🐠 FishLand Aquarium Website

This project is a real-world website developed for an aquarium shop to showcase their products and services online.

## 🌐 Features
- Responsive web design
- Home, About, Services, and Contact pages
- Clean and user-friendly interface
- Attractive layout with images and animations

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap

## 🎯 Objective
The objective of this project is to create an online presence for the aquarium shop, allowing customers to explore services, products, and contact the business easily.

## 🏪 About the Client
This website was developed for an aquarium shop to improve customer engagement and provide better accessibility to their offerings.

## 📁 Project Structure
- index.html
- about.html
- services.html
- contact.html
- assets (CSS, JS, Images)

## 🚀 How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser
